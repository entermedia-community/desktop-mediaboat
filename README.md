# desktop-mediaboat


# Electron

## Dependencies
`npm 6.14+`

## Install
```bash
npm install
```

## Run
```
npm start
```

## Create installers
Some installers require the origin OS to compile.

### Linux
```
npm run make
```

### Darwin
```
npm run makemac
```


### Windows
```
npm run makewin
```
